# Sistema de Leilões - LeiloesTDSat

## Descrição do projeto

Este projeto foi desenvolvido como atividade prática do curso Técnico em Desenvolvimento de Sistemas do Senac.

O sistema permite o gerenciamento de produtos de leilões. Atualmente, é possível cadastrar novos produtos informando nome e valor, consultar os produtos cadastrados por meio de uma tabela (JTable) e realizar a venda dos produtos cadastrados.

## Objetivo

Desenvolver um sistema para o gerenciamento de produtos de leilões, permitindo:

- Cadastro de produtos;
- Visualização dos produtos cadastrados;
- Venda de produtos.

## Tecnologias utilizadas

- Java
- MySQL
- Git
- GitHub
